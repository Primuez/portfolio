## Tech Stack

* Frontend: React
* Backend: Node.js with Express.js
* Database: MongoDB
* Authentication: JSON Web Tokens (JWT)
* API Documentation: Swagger
* Testing: Jest and Enzyme
* Linting: ESLint with Airbnb config
* Code Formatting: Prettier
* State Management: Redux

## Rules

* Use React for all frontend components
* Use Node.js with Express.js for all backend API endpoints
* Use MongoDB for all database operations
* Use JWT for all authentication and authorization
* Use Swagger for all API documentation
* Use Jest and Enzyme for all unit testing and integration testing
* Use ESLint with Airbnb config for all linting
* Use Prettier for all code formatting
* Use Redux for all state management
