# Contributing

Thank you for your interest in contributing! As this is a personal portfolio repository, unsolicited pull requests (especially those introducing new features or changing the design) may not be accepted. 

If you do submit a pull request (e.g., for fixing a typo or resolving a technical issue), you must confirm that:

1. The submission is entirely your own original work.
2. Your contribution is lawful and does not infringe on any third-party rights.
3. The submission is compatible with the proprietary terms set out in `LICENSE.txt` and does not attempt to subvert the protections on personal brand assets and content.

By submitting a pull request, you accept that your contributions will be bound by the same proprietary license terms governing this project.
