# Security Policy

## Supported Versions
Currently, only the latest version of this repository is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |

## Reporting a Vulnerability

I take the security of my systems seriously. If you discover a vulnerability or security issue within this repository or my live portfolio site, please practice responsible disclosure.

**How to report:**
Please reach out directly via my professional contact channels linked at [primuez.in](https://primuez.in). Do not open a public issue on GitHub for a security vulnerability. 

I will endeavor to acknowledge your report as quickly as possible and work towards a timely resolution.
